# Buckshot Roulette (霰弹枪轮盘赌)
基于C语言开发的经典霰弹枪轮盘赌小游戏，玩家将与恶魔展开回合制对决，通过策略性使用道具、选择射击目标，在充满随机性的子弹轮盘中存活并击败对手。

## 项目简介
本项目是一款复刻经典「霰弹枪轮盘赌」玩法的图形化小游戏（基于EasyX图形库开发），核心玩法围绕「实弹/空弹随机分布的弹匣」展开：
- 玩家与恶魔（Bot）轮流操控霰弹枪，选择向自己或对方射击；
- 不同关卡/回合对应不同的血量、子弹数量和道具配置；
- 可使用小刀、放大镜、饮料等道具改变战局，策略性提升获胜概率；
- Bot具备自主决策逻辑，会根据血量、子弹类型优先使用有利道具。

## 技术栈
- 开发语言：C语言
- 图形库：EasyX（用于图形化界面渲染、音效播放、交互）
- 编译环境：支持C99及以上标准的编译器（如MinGW、MSVC）

## 游戏规则

### 基础规则
- 霰弹枪子弹随机填充，每回合都是玩家先手，可以选择：
  1. **向自己开枪**
     - 若为真弹，则造成1点伤害，霰弹枪控制权交由敌方进行操作
     - 若为假弹，不造成伤害可再次进行选择
  2. **向敌人开枪**
     - 无论真假控制权都交由敌方
     - 若为真弹，则造成1点伤害
     - 若为假弹，不造成伤害
- 弹匣最多可填充八发子弹，子弹在回合开始短暂展示几秒，展示时随机排列，填充时随机填充。
- 道具随机抽取，每次开枪前都可使用，不限数量。未使用道具可继承，道具栏最多容纳8个道具，满则无法再抽取。

### 道具系统
| 道具ID | 道具名称 | 功能说明 |
|--------|----------|----------|
| 1      | 小刀     | 可使子弹伤害翻倍，若已使用则无法再次使用 |
| 2      | 放大镜   | 可查看当前子弹类型 |
| 3      | 饮料     | 退出当前子弹1发 |
| 4      | 手铐     | 可连续行动两次，若已使用则无法再次使用 |
| 5      | 烟       | 回复1点血量 |

### 关卡规则
共三关，子弹打完进入下回合，敌方血量归零进入下一关，我方血量归零则死亡。

#### 第一关
- 双方血量各为2点，无道具
- 第一回合：子弹为一真两假
- 第二回合：子弹为三真两假

#### 第二关
- 双方血量各为4点，每回合开始可抽取两个道具
- 第一回合：子弹一真一假
- 第二回合：子弹两真两假
- 第三回合：子弹三真两假
- 第四回合：子弹三真三假
- 第五回合：子弹五真两假
- 若还未结束则随机填充

#### 第三关
- 双方血量各为6点，每回合开始可抽取两个道具
- 第一回合：一真两假
- 第二回合：四真四假
- 第三回合：三真两假
- 第四回合：四真两假

## 核心功能
### 1. 关卡与回合系统
- 共3个核心关卡，不同关卡对应不同初始血量、道具数量：
  - 关卡1：玩家/恶魔初始血量2，无初始道具；
  - 关卡2：玩家/恶魔初始血量4，每回合开始可抽取2个道具；
  - 关卡3：玩家/恶魔初始血量6，每回合开始可抽取4个道具；
- 每个关卡分多个回合，回合数影响弹匣内实弹/空弹的数量，超出预设回合后子弹数量随机生成。

### 2. 子弹系统
- 弹匣容量上限8，包含「实弹（1）」和「空弹（2）」两种类型；
- 不同关卡/回合预设固定的实弹/空弹数量，超出后调用`randomBullet()`随机生成（实弹3~5发，空弹2~4发）；
- 子弹填充后会通过`shuffle()`随机打乱顺序，增加游戏随机性。

### 3. 道具系统
- 玩家和Bot均有8个道具栏位，关卡2和关卡3初始化时随机分配道具；
- Bot会自主判断道具使用时机（如低血量时优先用香烟回血、实弹时用小刀提升伤害）。

### 4. 战斗系统
- **玩家行动**：可选择向自己/恶魔射击，射击后根据子弹类型扣减对应角色血量，完成后控制权移交Bot（手铐道具可保留控制权）；
- **Bot行动**：自主使用道具（随机概率），并根据血量、子弹类型策略性选择射击目标（低血量时优先攻击玩家）；
- 子弹伤害默认1，小刀道具可临时提升至2，射击后重置为1。

## 快速开始
### 环境要求
1. 安装EasyX图形库（适配编译器：MinGW-w64 / Visual Studio 2019+）；
2. 编译器支持C99及以上标准；
3. 操作系统：Windows（EasyX仅支持Windows）。

### 编译与运行

1.  **项目入口：Buckshot Roulette.sln**
    这是 Visual Studio 的解决方案文件，双击即可在 Visual Studio中打开整个项目
    但是想要在Visual Studio中运行，需要配置EasyX图形库
3.  **编译结果：Buckshot Roulette.exe**
    项目编译后生成的可执行文件，路径：x64->release->Buckshot Roulette.exe
    直接双击即可运行霰弹枪轮盘赌游戏（无需打开 Visual Studio）。
